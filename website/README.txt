✅ 这是一个完全可以直接打开的纯HTML网站！

🎉 如何使用
───────────────────────────────────────────────────────────────

1. 双击 index.html 文件
2. 网站会在浏览器中打开
3. 所有链接都能正常工作！

就这么简单！

📤 部署选项
───────────────────────────────────────────────────────────────

方式1：直接使用
- 双击 index.html 即可在浏览器中打开
- 可以离线使用

方式2：上传到服务器
- 将 website 文件夹内容上传到服务器
- 通过域名访问

方式3：免费托管
- Netlify: 拖入 website 文件夹
- GitHub Pages: 推送到 GitHub
- Vercel: 部署静态文件

🛠️ 如需修改
───────────────────────────────────────────────────────────────

修改文章内容：
编辑 articles/*.html 文件

修改样式：
编辑 css/style.css 文件

添加新页面：
复制现有HTML文件，修改内容和链接


💡 提示
───────────────────────────────────────────────────────────────

1. 所有图片使用Unsplash链接，需要网络连接
2. 可以下载图片到本地，修改img标签的src路径
3. 表单提交需要配置后端（当前仅为展示）
4. 可以根据需要添加更多页面
